# iSorry.lol

iSorry.lol is the Django API for the iSorry.lol Client

## Development
Acitvate Virtual Environment
```bash
pipenv shell
```

Install dependencies
```bash
pipenv install
```

Run developement server on port 8080
```bash
python manage.py runserver 8080
```
